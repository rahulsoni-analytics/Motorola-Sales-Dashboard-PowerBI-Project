# # 📊 Motorola Sales Dashboard (Power BI | Data Analysis Project)

## 🔍 Project Overview
Built an interactive Power BI dashboard analyzing Motorola sales data with over **51M+ total sales**, **11M+ quantity sold**, and **1K+ transactions**.

The dashboard provides insights into city-wise performance, product trends, and customer behavior, helping businesses make data-driven decisions.

## 🛠 Tools Used
Power BI | Excel | SQL (Data Cleaning)

## 📌 Key Insights
- Identified top-performing cities contributing to **51M+ total revenue**
  
- Analyzed **11M+ units sold** to understand product demand patterns

- Evaluated customer transactions (**1K+ orders**) to track purchasing behavior

- Discovered sales trends across months to identify growth opportunities
